# PixelPenWords

Official website repository for **PixelPenWords** — a digital home for thoughtful tools, writing, product experiments, and creative growth.

Built as a personal platform to document learning, share projects, and create practical digital resources.

## About the Project

PixelPenWords is where I share:

- Digital tools and templates + beta test list
- Field Notes / reflections / lessons learned  
- Product design learning journey  
- Future experiments and resources  

## Built With

- [Astro](https://astro.build/)
- Tailwind CSS
- Markdown content collections

## Development

**Install dependencies:**
 `npm install`
**Run locally:**
 `npm run dev`
**Build production version:**
 `npm run build`

## Important Repository Structure Note
This repository contains the Astro project inside a nested folder:

ppw_website_astro_theme/

This happened during the early learning/build phase when the project was first created locally from a downloaded Astro theme before being connected to GitHub.

So if deploying (Cloudflare Pages / Vercel / Netlify), use:

Root Directory: ppw_website_astro_theme
Build Command: npm run build
Output Directory: dist

## Why It’s Structured This Way

During the initial setup, I began working locally with a zipped Astro theme and later connected that working folder directly to GitHub while learning repository structure and deployment workflows.

Rather than rebuild from scratch, I kept the existing structure and continued shipping.
